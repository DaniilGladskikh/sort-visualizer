# Визуализатор Алгоритмов Сортировки

Интерактивное веб-приложение для визуализации работы различных алгоритмов сортировки. Приложение позволяет наглядно увидеть, как работают популярные алгоритмы, сравнивать их эффективность и изучать принципы их действия. Создано с использованием современных веб-технологий: React, TypeScript и Vite.

## Возможности

*   **Визуализация алгоритмов**: Наглядное отображение процесса сортировки с цветовой индикацией сравнений и перестановок.
*   **Поддержка множества алгоритмов**:
    *   Сортировка пузырьком (Bubble Sort)
    *   Быстрая сортировка (Quick Sort)
    *   Сортировка слиянием (Merge Sort)
    *   Пирамидальная сортировка (Heap Sort)
    *   Сортировка вставками (Insertion Sort)
    *   Сортировка выбором (Selection Sort)
    *   Сортировка Шелла (Shell Sort)
    *   Шейкерная сортировка (Cocktail Shaker Sort)
    *   Гномья сортировка (Gnome Sort)
    *   Поразрядная сортировка (Radix Sort)
*   **Управление параметрами**:
    *   Выбор размера массива.
    *   Регулировка скорости анимации.
    *   Генерация нового случайного массива.
*   **Статистика**: Отображение количества сравнений, перестановок и времени выполнения.
*   **Адаптивный дизайн**: Приятный и современный интерфейс.

*   [React](https://react.dev/)

## Установка и запуск

1.  **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/DaniilGladskikh/sort-visualizer.git
    cd sort-visualizer
    ```

2.  **Установите зависимости:**

    ```bash
    npm install
    ```

3.  **Запустите режим разработки:**

    ```bash
    npm run dev
    ```

    Откройте [http://localhost:5173](http://localhost:5173) в вашем браузере.

## Сборка для продакшена

Для создания оптимизированной сборки выполните:

```bash
npm run build
```

Файлы сборки будут находиться в папке `dist`.
